# Student Stress Clustering Analysis

## Overview

This project analyzes a student lifestyle dataset to study how daily habits relate to stress levels and academic performance.

The analysis combines exploratory data analysis, Principal Component Analysis, outlier detection, model-based clustering, supervised classification, and regression-based mixture modeling.

## Goal

The goal of this project is to investigate whether student stress levels can be explained or reconstructed using variables such as:

- study hours
- physical activity hours
- sleep hours
- social hours
- extracurricular activity hours
- GPA

The project also evaluates how well clustering and classification methods can reproduce the original stress-level labels.

## Methods

The project includes:

- Exploratory data analysis
- Missing value inspection
- Variable renaming and preprocessing
- Principal Component Analysis
- Outlier detection using the IQR method
- Correlation heatmaps
- Model-based clustering with `mclust`
- Confusion matrix evaluation
- Classification Error Rate
- Supervised classification with `Rmixmod`
- Model comparison using CV and BIC
- Regression mixture modeling with `flexmix`
